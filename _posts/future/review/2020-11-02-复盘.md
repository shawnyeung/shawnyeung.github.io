## 1.1 可能的trend初期，pinbar即pb

　　如`6`和`41`

<br/>

---

<br/>

## 1.2 强劲趋势中，H1或L1可直接进场

　　`6`、`9`、`43`、`54`、`64` 均是进场机会

<br/>

---

<br/>

## 1.3 3push完成后，赶紧画出半分位作为pb的目标

　　如`4`到`10`，`37`到`53` 两个trend。半分位作为重要RS，可判断回撤力度的强弱。这两次都利索地突破了半分位，则下个目标位就有较大概率达成。

<br/>

---

<br/>

## 2.1 期待的setup没有出现，而是TR取而代之，则期待的setup可放弃

　　`11`到`15`的没有出现期望中的两腿回撤，而是重新回到开盘价，且从`15`开始出现连续多跟pinbar，说明此时已进入TR。两腿回撤的期待可以放弃。

<br/>

---

<br/>

## 2.2 明显的TR，头两次突破90%概率会fail

　　从`15`开始的TR，到`22`突破时，已有7根pinbar，TR的强大磁力会让突破极大概率成为假突破。`27`同样

<br/>

---

<br/>

## 2.3 若一个TR持续时间较长，2个小时左右，且出现至少两次假突破，则第三次突破成功的可能性大增，如果有FT，可入场

　　从`15`到`37`的TR持续了将近两小时，且出现两次假突破，且这个TR有完美的对称性（两次突破前后各有7根bar）。所以当`38`向下突破时，如果有强势的FT，则有较大概率会成功。

<br/>

---

<br/>

## 3.1 突破前高前低后，要立即画出RS

　　`23`和`40`和`51`都是如此，后面均出现回测

<br/>

---

<br/>

## 3.2 开盘1个半小时后，需对当天趋势类型有清醒认识

　　最早到`15`，基本断定当天是TR日。随后紧跟的tight TR更验证了观点。所以后面尽管出现了一次bear trend，则同样要期待会有较大概率回到开盘价。果真在`71`达成

<br/>

---

<br/>

## 3.3 出现HH或LL，及时更新通道线

　　当`44`突破`43`低点形成LL时，要持续更新下降通道，直到`51`<br/>
